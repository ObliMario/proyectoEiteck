API RESTful Documentación

Registro de Usuario (POST /api/usuarios)

Este endpoint registra un nuevo usuario con los datos proporcionados.

{
  "name": "Usuario Ejemplo",
  "email": "usuario@mail.com",
  "password": "passwordSeguro",
  "role": "user"
}

Inicio de Sesión (POST /api/auth/login)

Autentica al usuario y devuelve un token JWT para acceso a rutas protegidas.

{
  "email": "usuario@mail.com",
  "password": "passwordSeguro"
}

Modificar Usuario (PUT /api/usuarios/:id)

Permite actualizar la información del usuario. Solo accesible por el usuario correspondiente o por administradores. Solamente los administradores pueden modificar el rol del usuario

{
  "name": "Nuevo Nombre",
  "email": "nuevoemail@mail.com",
  "role": "user"
}

Listar Usuarios (GET /api/usuarios/)

Retorna una lista de todos los usuarios. Requiere token de autenticación (admin,user, guest).

Eliminar Usuario (DELETE /api/usuarios/:id)

Elimina el usuario especificado. Solo disponible para administradores.

Búsqueda de Usuario por Correo Electrónico (GET /api/usuarios/search?email={email})

Encuentra y muestra la información del usuario basada en el correo electrónico. Solo disponible para administradores.
